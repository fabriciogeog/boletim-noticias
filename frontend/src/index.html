Ajuda e Guia Rápido

Este é o guia de usuário principal da aplicação.

🎯 Características Principais

  • ✅ Coleta de Notícias via API GNews.io
  • 🤖 Sumarização Inteligente (Opcional) via Google Gemini
  • 🎙️ Geração de Áudio via gTTS (Padrão) ou ElevenLabs (Premium)
  • ♿ 100% Acessível com navegação por teclado
  • ⚡ Extremamente Leve - Roda em qualquer máquina.

🔑 Requisito Obrigatório: Chaves de API

Este sistema requer chaves de API para funcionar, que devem ser inseridas na aba Configurações.

  • GNews.io (Essencial): Para coletar notícias.
  • Google Gemini (Opcional): Para a sumarização com IA.
  • ElevenLabs (Opcional): Para a voz de alta qualidade.

🎮 Uso Básico (Interface Web)

  1. Configure: Insira suas chaves de API na aba "Configurações" e salve.
  2. Gere: Na aba "Gerar Boletim", marque as categorias e clique em "Gerar".
  3. Aguarde: ~10-20 segundos.
  4. Edite: Se necessário, clique em "Editar Texto" (Ctrl+E).
  5. Baixe: O MP3 gerado (Ctrl+D) para usar no seu programa!
